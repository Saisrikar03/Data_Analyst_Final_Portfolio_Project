DATA ANALYST FINAL PORTFOLIO PROJECT

T20 CRICKET PLAYER ANALYSIS & FINAL XI SELECTION


PROJECT OVERVIEW

* This project analyzes international T20 cricket player performance to identify the most effective and unbeatable Final XI using a data-driven approach.
* The focus is on selecting players who collectively can score at least 180 runs and defend 150 runs on average, based purely on performance metrics.

BUSINESS OBJECTIVE

* To eliminate subjective selection and use historical player data to build a balanced, high-impact cricket team by evaluating players across defined role-based criteria.

DATA SOURCE

* Player match data collected from ESPN

* Raw data extracted in JSON format using Bright Data

* Multiple JSON files processed and converted into structured CSV files

DATA PROCESSING

* Python (Jupyter Notebook) used for JSON extraction, validation, and CSV generation

* Power Query in Power BI used for data cleaning, transformation, and modeling

* Data organized by player roles such as Openers, Middle Order, Finishers, All-Rounders, and Bowlers

ANALYTICAL METHODOLOGY

* Defined clear role-based performance criteria for each player category

* Created advanced calculations using DAX for batting, bowling, and impact metrics

* Compared players using interactive visuals, trends, and benchmarks

* Selected the Final XI based on consistency, efficiency, and contribution

DASHBOARD HIGHLIGHTS

* Role-based player filtering and comparison

* Scatter plots for strike rate vs average analysis

* Individual player profile insights

* Final XI selection with combined team strength visualization

KEY OUTCOME

* The analysis delivers a data-backed Final XI that balances batting depth, bowling control, and all-round performance,
  demonstrating how analytics can support strategic sports decision-making.
